# Design patterns with TypeScript

## Creational patterns

- <b>Singleton</b> - only a single instance can exist.
- <b>Abstract Factory</b>
- <b>Builder</b> 
- <b>Factory Method</b> 
- <b>Prototype</b> 

## Available Scripts

In the project directory, you can run:

### `npm init -y`

### `npm i typescript -D`

Install typeScript

### `npm i -D ts-node`

Run typeScript code
